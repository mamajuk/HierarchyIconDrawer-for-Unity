## Overview

```HierarchyIconDrawer```는 **Hierarchy Window** 에서 특정 ```Component``` 가 부착된 ```GameObject``` 들을 아이콘으로 직관적으로 식별할 수 있도록 도와주는 유틸리티입니다. 에디터 메뉴창의 **Utility > HierarchyIconDrawer Settings** 메뉴에서 최대 32개까지의 아이콘을 설정할 수 있습니다.
